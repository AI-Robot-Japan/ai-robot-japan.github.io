# AI Robot Japanとは

**AI Robot Japan**は、AIロボットを中核に、研究・開発から社会実装までを一気通貫で推進するためのコミュニティです。

生成AI・機械学習の進展により、認識・推論・計画・制御が統合されつつある今、ロボットを動かすための実装知と産業知を結集し、日本の競争力強化(国力・産業基盤の再構築)に資するエコシステム形成を目指します。

日本では、ハードウェア人材とAI/ソフトウェア人材が分断され、知見や人材が領域内に閉じがちです。

本コミュニティはそれらを横断的に接続し、データ・モデル・ハードウェアを統合した開発体制(R&D-PoC-プロダクト化)を加速させる「融合の場」を提供します。

米中を中心に国際競争が激化するなか、日本が一丸となってAIロボットを推進するための、実務者ネットワーク/共創基盤として機能します。

---

# 大事にしている雰囲気

**AI Robot Japan**が何より大事にするのは、AIロボットの「つくって動かす」体験から生まれるワクワク感です。最先端の技術を追うだけでなく、実機・デモ・プロトタイピングといったものづくりを通じて、学びと熱量が自然に循環する場を目指しています。

また、AIロボットはソフトウェア、ハードウェア、製造、研究、事業など多領域の結節点にあります。だからこそ本コミュニティは、異業種・異分野の人が混ざり合い、偶発的な出会いと交流が生まれる「多様で開かれた場」であることを重視します。

私たちは、排他的・クローズドなコミュニティを志向しません。基本スタンスはオープン。参加者同士がリスペクトを持って関わり合い、初心者から第一線の実務者までが安心して議論し、協力し、挑戦できる、そんな楽しくて、温かくて、ワクワクするコミュニティをつくっていきます。

# 取り扱うテーマ例

- **AI**：模倣学習/強化学習/自己教師あり学習、基盤モデル、マルチモーダル
- **自律性**：タスク・モーションプランニング、行動生成、SLAM、ナビゲーション
- **制御・実装**：リアルタイム制御、状態推定、シミュレーション（Sim2Real）、デジタルツイン
- **開発基盤**：ROS/ROS 2、エッジAI、MLOps/LLMOps、センサフュージョン、安全・信頼性
- **産業応用**：製造・物流・点検・医療介護などの現場実装、事業化・規格/標準化動向

# 想定する活動内容

- 勉強会/ハンズオン(実機持ち込み、開発環境セットアップ、実装レビュー)
- ハッカソン(特定テーマでの短期集中開発、デモデイ)
- 年1回のカンファレンス(研究×産業×政策の接続、技術潮流の整理と共有)
- 情報発信(ポッドキャスト等で最新のフィジカルAI動向・論文/事例の解説)
- オフライン交流(領域横断の人材マッチング、共同研究・共同開発の起点づくり)

# 開催エリア

東京に限らず、製造業・ロボット産業の集積を踏まえ、大阪・京都・名古屋などでも開催を検討し、全国規模の共創ネットワークを構築します。
